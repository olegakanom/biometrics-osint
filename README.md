# Поиск по фото с лицом (OSINT 2026)

Актуальная подборка OSINT-инструментов для поиска по фотографии с лицом: профили в соцсетях (VK, OK, TikTok, Clubhouse и др.), похожие фото, клоны, вебкам-модели, бары/клубы, удалённые изображения, телефоны, почты, документы + улучшение качества фото и проверка дипфейков. OSINT 2026

Наш дискорд канал (пробивы по ФИО, почтам, адресу, аккаунтам в соцсетях, крипто и т.д.) — https://discord.gg/zDNW39HKva

## Основные инструменты поиска по лицу

| №  | Сервис / Сайт / Бот                                      | Что находит / Основные возможности                                                                 |

| 5  | https://facecheck.id/                                    | Лицо в соцсетях, ссылки на источники (среднее качество, альтернативы: lenso.ai, profacefinder.com)  |

Советы:
- Лучшие для лиц в 2026: PimEyes, Lenso.ai, Yandex Images (особенно для России/СНГ), FaceCheck.ID, ProFaceFinder, Search4Faces (для VK/OK/TikTok).
- Многие сервисы (PimEyes, FaceCheck) требуют VPN, особенно из РФ — блокировки и гео.
- Для русского сегмента комбинируй Search4Faces + FindClone_bot + vk.watch.
- Улучшай фото перед поиском (Remini/CodeFormer) — точность растёт в разы.
- Проверяй актуальность: боты часто падают, сайты меняют базы/политику.
